ซิงก์รูปนักเรียนจาก Google Drive

สามารถแยกรูปภาพเป็นโฟลเดอร์ย่อยรายห้องเรียนได้ ระบบจะเข้าไปสแกนหาจากทุกโฟลเดอร์ข้างในเอง

ขั้นตอนการตั้งค่า Google Apps Script (ทำครั้งเดียว):

  1. สร้างโฟลเดอร์รูปภาพบน Google Drive ตั้งชื่อไฟล์รูปภาพเป็นรหัสนักเรียน (เช่น 6032.jpg)
  2. แชร์โฟลเดอร์ให้เป็น "ทุกคนที่มีลิงก์สามารถดูได้" (Anyone with the link can view)
  3. เปิดเว็บ script.google.com สร้างโปรเจกต์ใหม่
  4. คัดลอกโค้ด Apps Script (ในไฟล์สรุป) ไปวาง แล้วกดบันทึก
  5. กด "Deploy" (การทำให้ใช้งานได้) -> เลือก "New deployment" -> ชนิด "Web app"
  6. ตั้งค่า: Execute as = "Me", Who has access = "Anyone"
  7. กด Deploy แล้วคัดลอกลิงก์ Web App URL มาใส่ช่องด้านบน
